Data & Integritet

Hur SVERA hanterar data, personuppgifter och AI-bearbetning.

Enbart öppen, publik data

SVERA samlar inte in personuppgifter. All information på denna sajt hämtas från öppna, publika källor som är tillgängliga för alla:

  • WebTracking.se — öppen GPS-spårning och resultat från svenska offshoretävlingar
  • SVEMO TA — Svemos officiella tävlingsadministration (kräver inloggning — ej öppen för allmänheten, används enbart för kalender och tävlingsdata)
  • UIM Sport — internationella tävlingskalendrar, öppet publicerade
  • Nyhetskällor — publika artiklar från Powerboat Racing World, F1H2O och Powerboat News

Vi använder inga cookies, ingen analytics, ingen spårning och inga tredjepartsskript för datainsamling. Vi har inget användarregister och du behöver inte logga in för något.

Personnamn — verifiering mot öppna källor

SVERA visar enbart namn som redan är publicerade på öppna plattformar. Innan ett namn visas på svera.nu kontrollerar vi att personen förekommer på en öppen källa — exempelvis WebTracking.se, som är helt öppet för alla.

Om ett namn inte finns publicerat officiellt på någon öppen plattform visas det inte på SVERA. I de fall data inte är officiellt bekräftad används platsinnehållare (placeholders) istället för riktiga namn.

AI-bearbetning

För att sammanfatta nyheter och bearbeta text använder SVERA två AI-modeller:

  • DeepSeek — används för allmänna sammanfattningar. Hanterar enbart anonymiserade förfrågningar via POST utan datalagring, men är inte ZDR-certifierad
  • Qwen (ZDR) — används vid behov när personnamn eller känslig data kan förekomma. Körs via ZDR-godkänt API (Zero Data Retention) — ingen data sparas över huvud taget. Högre kostnad per anrop

I praktiken innebär detta:

  • Personuppgifter skickas aldrig till DeepSeek
  • När namn behöver bearbetas används antingen Qwen (ZDR) eller platsinnehållare som ersätts lokalt
  • Ingen AI-leverantör lagrar frågor eller svar från SVERA

Vi säljer och delar aldrig data

SVERA säljer inte, delar inte och distribuerar inte någon data till tredje part. Punkt.

All information på svera.nu är publikt tillgänglig — precis som källorna den hämtas ifrån. Om någon extern part väljer att skrapa, curla eller på annat sätt samla in data från vår sajt är det inte vi som säljer eller delar den. Det är utomstående som dränerar öppet publicerad information — och det är bortom vår kontroll, precis som det är med vilken publik webbplats som helst.

Vi uppmanar alla som använder data från svera.nu att göra det ansvarsfullt och med respekt för de personer som förekommer.

Vill du bli borttagen?

Om du hittar ditt namn på svera.nu och vill att det tas bort — kontakta oss så raderar vi det så snart vi kan. Vi jobbar inte dygnet runt, men vi prioriterar borttagningar och hanterar dem så fort det är möjligt. Inga frågor, inget krångel.

Vi respekterar din önskan fullt ut. Vår sajt är till för att göra båtracinginfo tillgänglig, inte för att störa någon.

Sammanfattning

  • Vi samlar ingen persondata från besökare
  • Vi använder inga cookies eller spårning
  • All data kommer från öppna, publika källor
  • Namn visas bara om de redan är officiellt publicerade
  • AI-bearbetning sker via DeepSeek (allmänt) och Qwen ZDR (persondata, namn)
  • Vi säljer eller delar aldrig någon data
  • Borttagning sker så snart som möjligt på begäran (vi jobbar inte dygnet runt, men vi prioriterar det)

Vi finns här för båtracingen — inte mot den.
